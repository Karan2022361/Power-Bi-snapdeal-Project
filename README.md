# Power BI Snapdeal Analytics Project 

## Overview

This version of the Snapdeal Analytics project presents **6 analytical tasks documented separately** and **all dashboards combined into a single Power BI (.pbix) file**. The project demonstrates structured analysis, correct visual selection, and business-focused insights using Power BI.

---
---

## Tools Used

* Power BI (DAX, visuals, data modeling)

---

## Task Summary & Files

### Task 1: Discount vs Price Relationship

* **Visual Used:** Scatter Plot
* **Insight:** Discounts are spread across all price ranges; no strong linear relationship.

### Task 2: Conditional Formatting

* **Visual Used:** Table
* **Insight:** Highlights high discounts and low-rated products for quick analysis.

### Task 3: Average Price vs Average Rating by Subcategory

* **Visual Used:** Column & Line Combo Chart
* **Insight:** Higher-priced subcategories do not consistently have higher ratings.

### Task 4: Discount vs Customer Rating

* **Visuals Used:** Column Chart + Scatter Plot
* **Insight:** Higher discounts do not guarantee higher customer ratings.

### Task 5: Average Discount Trend Over Time

* **Visual Used:** Line Chart
* **Insight:** Average discount shows an increasing trend over scraping periods.

### Task 6: KPI Business Interpretation

* **KPIs:** Avg Price ₹950 | Avg Discount 40% | Avg Rating 3.8
* **Insight:** Balanced pricing and discount strategy with good customer satisfaction.

## Power BI Dashboard

* **Single Dashboard File:**

* Contains all visuals corresponding to Tasks 1–5
* Task 6 is interpretation-only and does not require a dashboard
